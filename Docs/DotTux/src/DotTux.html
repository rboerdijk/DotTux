DotTux

This is the technical documentation set for DotTux version 1.1.

Target Audience

This technical documentation set is intended for Tuxedo developers who want to develop Tuxedo clients and servers in C# using DotTux. Readers are expected to know how to:

  • Install and configure Tuxedo.
  • Develop Tuxedo clients and servers in C.
  • Install the .NET Framework.
  • Work with .NET assemblies and the global assembly cache.
  • Develop applications in C#.

Getting started

To get started developing Tuxedo clients and servers in C#:

  1. Install DotTux as described in the DotTux Installation Guide
  2. Get familiar with the DotTux API by studying the DotTux Programming Reference.
  3. Develop and run your own .NET based Tuxedo clients and servers using the DotTux.dll assembly and the DotServer executable.

Documentation Index

The DotTux documentation set consists of the following documents:

  • DotTux Installation Guide
    Describes how to install DotTux on a server machine in your local Tuxedo environment.
  • DotTux Programming Reference
    Describes the DotTux API and illustrates how to use it by means of a number of sample Tuxedo clients and servers written in C#.
  • DotTux.dll Manual Page
    Describes how to compile Tuxedo clients and servers written in C# and how to run .NET based Tuxedo clients using the DotTux.dll assembly.
  • DotServer Manual Page
    Describes how to run .NET based Tuxedo servers using the DotServer executable.
