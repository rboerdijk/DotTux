DotTux Workstation

This is the technical documentation set for DotTux Workstation version 1.1.

Target Audience

This technical documentation set is intended for Tuxedo developers who want to develop Tuxedo/WS clients in C# using DotTux Workstation. Readers are expected to know how to:

  • Install and configure Tuxedo/WS.
  • Develop Tuxedo/WS clients in C.
  • Install the .NET Framework.
  • Work with .NET assemblies and the global assembly cache.
  • Develop applications in C#.

Getting started

To get started developing Tuxedo/WS clients in C#:

  1. Install DotTux Workstation as described in the DotTux Workstation Installation Guide
  2. Get familiar with the DotTux Workstation API by studying the DotTux Workstation Programming Reference.
  3. Develop and run your own .NET based Tuxedo/WS clients using the DotTuxWS.dll assembly.

Documentation Index

The DotTux Workstation documentation set consists of the following documents:

  • DotTux Workstation Installation Guide
    Describes how to install DotTux Workstation on a client machine in your local Tuxedo environment.
  • DotTux Workstation Programming Reference
    Describes the DotTux Workstation API and illustrates how to use it by means of a number of sample Tuxedo/WS clients written in C#.
  • DotTuxWS.dll Manual Page
    Describes how to compile and run Tuxedo/WS clients written in C# using the DotTuxWS.dll assembly.
