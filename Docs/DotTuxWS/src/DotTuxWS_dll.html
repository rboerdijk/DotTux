DotTuxWS.dll

DotTuxWS.dll is the .NET assembly containing the .NET API for Tuxedo/WS described in the DotTux Workstation Programming Reference. The DotTuxWS.dll file is located in the bin subdirectory of the DotTux Workstation directory.

Compiling DotTux Workstation client using DotTuxWS.dll

To compile a DotTux Workstation client using DotTuxWS.dll, simply add the DotTuxWS.dll file as a as a library reference to the compiler. For example, the command for compiling the SimpleClient class written in C# looks like this: 

csc /reference:DOTTUXWSDIR\bin\DotTuxWS.dll /out:SimpleClient.exe SimpleClient.cs
where DOTTUXWSDIR is the DotTux Workstation directory.

Running DotTux Workstation clients using DotTuxWS.dll

To run a DotTux Workstation client using the DotTuxWS.dll assembly, the DotTuxWS.dll assembly must either be installed in the global assembly cache or copied to the application directory (i.e., the directory containing the client executable). Furthermore, the DotTuxWS.dll assembly requires the following environment variables to be set at runtime:

PATH
Must be set to include the bin subdirectory of the DotTux Workstation directory. For example: 
set PATH=%PATH%;C:\OTPSystems\DotTuxWS-1.1.0\bin
These environment variables must be set in addition to the Tuxedo system environment variables described in the Tuxedo installation guide (Chapter 6 "Performing Post-Installation Tasks", section "Setting Environment variables on a Windows System").
