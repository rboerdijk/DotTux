DotTux Workstation Installation Guide

This document describes how to install DotTux Workstation on a client machine in your local Tuxedo environment.

» Prerequisites
» Installing the DotTux Workstation software package

Prerequisites

Before you start the installation, make sure the following prerequisites are met:

  1. Tuxedo/WS version 8.1 is installed on the target machine.
  2. The .NET Framework version 1.1 or higher is installed on the target machine. The .NET Framework can be downloaded from http://msdn.microsoft.com.
  3. You have the DotTux Workstation software package DotTuxWS-1.1.x-win32.zip, where x is the latest available patch level for DotTux Workstaiton 1.1.

Installing the DotTux Workstation software package

The DotTux Workstation software package can be installed by simply unpacking it in a local directory on the target machine. This will create a new subdirectory in the selected installation directory containing the DotTux Workstation software components. For example, if you unpack the DotTuxWS-1.1.x-win32.zip software package in the directory c:\OTPSystems then a new DotTux Workstation directory c:\OTPSystems\DotTuxWS-1.1.x will be created.
